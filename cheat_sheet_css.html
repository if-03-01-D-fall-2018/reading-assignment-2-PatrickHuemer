World Wide Web General

How are resources (html pages, images, etc) addressed in the web?


URL stands for Uniform Resource Locator. A URL is nothing more than the address of a given unique resource on the Web.

What are the general components of a url?


http is the protocol. The first part of the URL indicates which protocol the browser must use.
www.example.com is the domain name. It indicates which Web server is being requested.
:80 is the port. It indicates the technical "gate" used to access the resources on the web server.
/path/to/myfile.html is the path to the resource on the Web server.
?key1=value1&key2=value2 are extra parameters provided to the Web server.
#SomewhereInTheDocument is an anchor to another part of the resource itself.

What are the most essential technologies that made the World Wide Web such a success?


URL: Uniform Resource Locator,is a system of globally unique identifiers for resources on the Web and elsewhere.
HTML: HyperText Markup Language, is the publishing language.
HTTP: HyperText Transfer Protocol, is the foundation for the WWW to communicate with data.

Web Server

What is a web server?


A web server is a server that sends documents to clients such as Web browser transmits. A web server is the computer with web server software or only the web server software itself.
